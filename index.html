My 4-Day Learning Journey

1

Day 1: Web Fundamentals

Foundation Building

Started with the core concepts of frontend development - understanding how HTML, CSS, and JavaScript work together to create structured and interactive web pages.

  • HTML structure and semantics
  • Metadata and the <head> tag
  • CSS color models including RGBA and HSLA
  • Basic page structure and organization
2

Day 2: CSS Mastery

Styling & Layout

Deepened my CSS knowledge by learning about the box model, positioning elements effectively, working with typography, and creating gradients and backgrounds for enhanced visual appeal.

  • CSS Box Model fundamentals
  • Element positioning (static, relative, absolute, fixed)
  • Typography principles and web fonts
  • Creating gradients and background effects
  • Layout techniques and spacing
3

Day 3: Content & Structure

Advanced HTML

Learned to structure and present content better by exploring lists, embedding media using video, audio, and iframes, designing forms, organizing data in tables, and writing cleaner, more efficient HTML and CSS code.

  • Working with different list types
  • Embedding media (video, audio, iframes)
  • Form design and input types
  • Data tables and organization
  • Writing semantic and accessible code
4

Day 4: Tools & Workflow

Developer Environment

Focused on practical tools and workflows - exploring VS Code fundamentals, extensions like Prettier, Live Server, and ESLint, along with Git basics such as initializing repositories, writing meaningful commits, and managing version control efficiently.

  • VS Code setup and customization
  • Essential extensions for web development
  • Git basics and repository management
  • Version control workflows
  • Code formatting and quality tools

Key Takeaways

These four days have strengthened both my technical understanding and my ability to apply best practices in real-world web development tasks. I've gained confidence in creating structured web pages, styling them effectively, and using professional development tools.
